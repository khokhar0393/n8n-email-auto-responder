# Email Auto-Responder System

## Overview

This n8n workflow automatically captures lead information, stores it in Google Sheets, notifies the team via email, and sends an automatic confirmation email to the customer.

## Tools Used

* n8n
* Google Sheets
* Gmail
* Webhooks

## Workflow

Webhook → Google Sheets → Team Notification Email → Customer Auto Reply Email

## Features

* Automatic lead capture
* Google Sheets integration
* Team email notifications
* Customer auto-response emails
* Real-time workflow automation

## Use Case

Ideal for businesses and agencies that want to automate lead handling and instantly acknowledge customer inquiries.
